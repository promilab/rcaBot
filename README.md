# RCA Bot Repository

This is the accompanying repository to the paper: "AI-augmented Root Cause Analysis in Process Mining".
The repository contains an overview of generated responses from the implemented prototype, technical details on the implementation and the transcripts of the expert interviews. 

- **[Proof_of_Concept_Results](./results)** - Results from the Single-factor and Multi-factor RCA
- **[Expert_Interview_Transcripts](./Expert_Interview_Transcripts)** - Transcripts of the expert interviews
- **[docs/DEVELOPER_GUIDE.md](./docs/DEVELOPER_GUIDE.md)** - Architecture and system design
